# Bienvenido a la Documentación de Didi Food

Esta documentación cubre todos los microservicios que componen la plataforma Didi Food.

##  Comenzar Rápidamente

- **[Guía Rápida](/guides/)** - Integración paso a paso
- **[Autenticación](/auth/)** - Registro, login y tokens JWT
- **[Gestión de Usuarios](/users/)** - Administración de usuarios
- **[Productos](/products/)** - Catálogo de productos
- **[Pedidos](/orders/)** - Gestión de pedidos

## Flujo Típico

1. **Registrar usuario** en el servicio de Autenticación
2. **Obtener token JWT** mediante login
3. **Consultar productos** disponibles
4. **Crear pedido** con los productos seleccionados
