# Guía Rápida

Esta guía te muestra cómo iniciar el proyecto y ejecutar los microservicios.

## Pasos iniciales

1. Clona el repositorio.
2. Instala las dependencias con `npm install`.
3. Ejecuta el proyecto con `npm run dev`.
